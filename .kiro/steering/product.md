# Product Overview

Shower Tile Layout Visualizer - A web-based tool for planning and visualizing tile layouts in shower installations.

## Purpose

Helps contractors, DIYers, and homeowners calculate tile requirements and visualize tile patterns for three-wall shower installations (back wall, left wall, right wall).

## Key Features

- Visual tile layout generation with SVG rendering
- Multiple tile patterns (brick/running bond 50%, 1/3 offset, straight stack)
- Automatic cut tile calculations with precise measurements
- Ledger board installation support (bottom row installed last)
- Tile numbering for installation sequence
- Material estimation with 10% waste factor
- Fractional inch display (1/8", 1/4", etc.) for contractor-friendly measurements
- Real-time updates on input changes
